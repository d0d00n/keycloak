Keycloak 8.x-1.x
----------------

Description
-----------
This module alters the default search box in many ways.
If you need to have options available like in advanced search,
but directly in the search box, this module is for you.

Install
-------
* Install the module by either adding it to your composer dependencies using
  composer require "drupal/keycloak:^1.0"
  or downloading the module and manually extracting it to your modules folder.
  For manual install, make sure to also install the openid_connect module.
* Enable the module.
* Go to the openid_connect settings and enable the Keycloak client.
